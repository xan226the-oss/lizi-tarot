# 粒子塔罗牌

一个以原创 78 张塔罗牌为基础的 Next.js 互动体验。它提供牌库浏览、常规与手势抽牌、象征性 AI 综合解读，以及大阿尔卡纳人格单聊。

> 塔罗内容仅作象征性陪伴与自我观察，不构成医疗、法律、财务或其他专业建议。

## 当前状态

本仓库是可运行的源代码发布版本：全部 78 张运行时卡牌资源已经过完整性校验，单聊界面包含可降级的静态星空与桌面指针粒子反馈。

这是开源项目：代码采用 MIT 许可证，原创卡牌插画与文字采用 CC BY 4.0。两者均允许商业使用、修改和再分发；素材再使用必须保留署名、许可证链接并标注修改。

## 功能

- 原创 78 张塔罗牌库：支持检索、筛选、逐牌阅读和场景化内容。
- 常规抽牌与手势抽牌：支持牌阵流程、正逆位和本地结果呈现。
- AI 综合解读：在配置 DeepSeek 密钥后，为已有牌阵生成克制的连续解读。
- 大阿尔卡纳人格单聊：每张可聊天的大阿尔卡纳牌拥有独立人格边界、输入校验、速率限制与安全回退。
- 沉浸式聊天画布：静态 CSS 星空始终可用；粒子效果仅在精确指针且未开启“减少动态效果”时启用。

## 本地运行

需要 Node.js 22 或更高版本。

运行：`npm install`，复制 `.env.example` 为 `.env.local`，然后执行 `npm run dev`。

访问 `http://127.0.0.1:3000`。`.env.local` 仅供本机使用，绝不能提交到 Git。

### 环境变量

| 变量 | 用途 |
| --- | --- |
| `DEEPSEEK_API_KEY` | 启用 AI 综合解读和人格单聊。未设置时，相关接口应返回安全的公开错误，不暴露配置细节。 |
| `DEEPSEEK_MODEL` | 常规牌阵综合解读使用的模型。 |
| `DEEPSEEK_CHAT_MODEL` | 大阿尔卡纳人格单聊使用的模型。 |
| `DATABASE_URL`、`NEXT_PUBLIC_SUPABASE_URL`、`NEXT_PUBLIC_SUPABASE_ANON_KEY`、`SUPABASE_SERVICE_ROLE_KEY` | 为数据库或 Supabase 集成预留；部署前仅在实际启用对应能力时配置。 |

## 验证

依次执行：`npm test`、`npx tsc --noEmit`、`npm run lint`、`git diff --check`，以及 `npm run tarot:verify -- --ids 1-78 --strict --runtime-only`。

共享开发预览运行时，不要同时执行 `npm run build`，以免干扰共享的 `.next` 目录。
